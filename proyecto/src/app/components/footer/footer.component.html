Esta aplicación ha sido desarrollada por Alejandra Carrasquilla Rodríguez-Córdoba, alumna del IES Trassierra, como proyecto educativo para fomentar la lectura colaborativa.

Ayuda

Si experimentas problemas críticos durante el uso de la aplicación (como errores al guardar libros, problemas de sesión o fallos en la visualización de reseñas), por favor recarga la página e intenta nuevamente. Para soporte adicional, contacta con el profesor responsable del proyecto.
